你可以使用 [Zeabur](https://zeabur.com) 来部署你的 Astro 网站，Zeabur 支持 SSR 和静态站点。

本指南包含通过网站 UI 部署到 Zeabur 的说明。

## 项目配置

你的 Astro 项目可以以两种不同的方式部署到 Zeabur：静态站点或服务器渲染站点。

### 静态站点

你的 Astro 项目默认是一个静态站点。你不需要任何额外的配置来将静态 Astro 站点部署到 Zeabur。

### 服务器渲染站点

1. 在你的 `astro.config.mjs` 文件中添加以下内容：

```js
import node from '@astrojs/node';

export default defineConfig({
  output: 'server',
  adapter: node({
    mode: "standalone"
  })
})
```

2. 修改你的 `package.json` 文件中的 `start` 命令：

```json
{
  "scripts": {
    "start": "node dist/server/entry.mjs"
  }
}
```

## 如何部署

你可以通过网站 UI 部署到 Zeabur。

### 网站 UI 部署

你可以将你的 Astro 站点部署到 Zeabur，如果你的项目存储在 GitHub 上。

1. 在 [Zeabur 控制台](https://dash.zeabur.com) 点击 <kbd>创建新项目</kbd> 

2. 配置 GitHub 安装并导入仓库。

3. Zeabur 将自动检测你的项目是否为 Astro 项目，并使用 `astro build` 命令构建它。

4. 构建完成后，你可以绑定一个域名到你的站点并访问它。

当你的项目被导入并部署后，所有对分支的推送都会自动生成新的构建。

你可以在 [Zeabur 文档](https://zeabur.com/docs/get-started/) 中了解更多关于部署的内容。
